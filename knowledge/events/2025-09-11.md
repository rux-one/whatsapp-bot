## Lokalny agent kodowania (działający!)

**VS Code + Cline extension**

Tryby: `plan` / `act`  
Reguły: `.clinerules/*.md` — przykłady: https://github.com/cline/prompts/blob/main/.clinerules

Konfiguracja:
* API Provider: `Ollama`
* Model: `hf.co/unsloth/Qwen3-Coder-30B-A3B-Instruct-GGUF:UD-Q4_K_XL`
* Context Window Size: `32768` (optymalnie dla 24 GB VRAM)
* Request Timeout: `15000 ms`

## Lokalne autouzupełnianie

**VS Code + Ollama Autocoder**

```json
"ollama-autocoder.endpoint": "http://localhost:11434/api/generate",
"ollama-autocoder.model": "qwen2.5-coder:14b",
"ollama-autocoder.prompt window size": 32768
```

Użycie: `Command Palette → Autocomplete with Ollama`
